# 坐骑

坐骑是玩家角色养成的独立子系统。玩家等级达到 10 级后自动开启，获得 0 星一阶坐骑「青角牛」。坐骑拥有品阶和星级两条成长线。

## 命令

```
坐骑
坐骑升星
一键升星
坐骑进阶 数量
一键进阶
确认进阶 数量
确认一键进阶
坐骑帮助
显化·东方青龙
显化·西方白虎
显化·南方朱雀
显化·北方玄武
```

## 玩法规则

- 玩家等级 ≥ 10 自动开启坐骑模块。
- 一阶至十阶坐骑共用升星物品「兽血」和进阶物品「兽魂丹」。
- 显化坐骑使用「幻兽血」升星、「幻兽魂丹」进阶。
- 极显化坐骑使用「极幻兽血」升星，无法进阶。
- 满星后才能进阶，进阶概率公式：P = (当前进阶祝福值 / 进阶祝福总值) ^ 3。
- 进阶祝福值 24 小时有效，到期未进阶则清零。
- 十阶满星进阶成功后选择显化方向（四选一），选择不可更改。
- 显化坐骑满星进阶后变为极显化坐骑，极显化坐骑只能升星。
