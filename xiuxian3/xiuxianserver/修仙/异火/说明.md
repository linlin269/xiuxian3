# 异火组件

## 概述

异火独立成体系，不复用纳戒物品、背包物品或武器实例。共 23 种异火，排名、名称、加成、外观和效果固定落库。

## 命令入口

| 命令 | 作用 |
|---|---|
| 异火 | 查看自己持有异火、当前装备异火和入口按钮 |
| 异火列表 | 查看 23 种异火列表、排名和倍率 |
| 异火详情 异火名 | 查看单个异火外观、效果、来源和倍率 |
| 异火装备 异火名 | 装备已拥有异火 |
| 异火卸下 | 卸下当前异火，攻击倍率回到 1.0 |
| 异火合成 | 尝试合成帝炎 |
| 异火交易 | 展示异火交易说明，提示使用二手市场上架和购买 |

## 数据库表

- `flame_defs`：23 种异火定义。
- `player_flames`：玩家持有异火，`(client_id, flame_id)` 复合主键。
- `flame_fusion_records`：帝炎合成记录。

## 掉落规则

- rank 21~23 可在任意区域探险结算获得（极低概率）。
- rank 2~23 可在首领奖励或虫洞奖励掉落。
- rank 1 帝炎不掉落，只能由 22 种异火合成。
- 同一玩家每种异火最多持有 1 个；重复掉落转补偿。

## 合成规则

- 检查 rank 2~23 的 22 种异火。
- 缺失任意一种时不消耗已有异火，返回缺失列表。
- 全部满足时删除 rank 2~23 的持有记录，插入帝炎并自动装备。

## 二手市场

- `item_type = flame`，`item_id = flame_id`，`quantity = 1`。
- 已装备异火不能上架。
- 购买时检查买家是否已有同名异火或已有帝炎。
