# 镇渊诛邪说明

镇渊与诛邪是一条独立成长线：镇渊提供固定基础值，诛邪只放大镇渊基础值，并额外叠加恢复、探险、黑市购买折扣。

## 命令

```text
镇渊 / 诛邪
镇渊帮助 / 诛邪帮助
镇渊面板 / 诛邪面板
镇渊积分 / 诛邪积分
```

## 组件关联

- 上游读取：[`xiuxian3/xiuxianserver/修仙/sql.py`](../sql.py) 的玩家与积分日志表、[`xiuxian3/xiuxianserver/修仙/common.py`](../common.py) 的角色重算入口、[`xiuxian3/xiuxianserver/修仙/探险/service.py`](../探险/service.py) 的普通怪结算、[`xiuxian3/xiuxianserver/修仙/wormhole_service.py`](../wormhole_service.py) 与 [`xiuxian3/xiuxianserver/修仙/首领/service.py`](../首领/service.py) 的 Boss 击杀结果。
- 下游写入：玩家镇渊诛邪成长表、积分日志表、玩家面板摘要、休息缩时、探险缩时、黑市购买折扣。
- 被消费结果：[`xiuxian3/xiuxianserver/修仙/common.py`](../common.py) 在人物基础属性重算时消费镇渊/诛邪追加值；[`xiuxian3/xiuxianserver/修仙/玩家/service.py`](../玩家/service.py) 在玩家面板和休息逻辑中消费恢复加成；[`xiuxian3/xiuxianserver/修仙/探险/service.py`](../探险/service.py) 消费探险缩时和普通怪积分接口；[`xiuxian3/xiuxianserver/修仙/黑市/service.py`](../黑市/service.py) 消费黑市购买折扣；[`xiuxian3/xiuxianserver/修仙/wormhole_service.py`](../wormhole_service.py) 与 [`xiuxian3/xiuxianserver/修仙/首领/service.py`](../首领/service.py) 消费 Boss 积分接口。
