# 英灵殿模块说明

## 功能概述
英灵殿是 PvE 挑战模块，每小时自动刷新 10 名 NPC 精英怪，玩家可单人挑战。

## 命令

| 命令 | 说明 |
|------|------|
| 英灵殿 | 查看英灵殿概览 |
| 英灵殿列表 / 英灵 | 查看当前批次 NPC 列表 |
| 挑战英灵 NPC名 | 挑战指定 NPC |
| 英灵殿记录 | 查看个人挑战记录 |

## 核心机制

- **刷新**：每小时整点自动刷新 10 名 NPC
- **数值**：固定基值 + 首领公式 × 稀有度比例，8级稀有度（R→UR）
- **战斗**：boss_panel=True，行动上限 80 次
- **奖励**：镇渊积分 + 恢复药品 + 经验（升级所需 0.5%）

## 扩展入口

| 调整目标 | 改什么 |
|----------|--------|
| NPC 数量 | `HALL_NPC_COUNT` |
| 刷新频率 | 定时任务 cron 表达式 |
| 稀有度分布 | `NPC_TIER_WEIGHTS` |
| NPC 整体强度 | 固定基值公式中的乘数 |
| 经验比例 | `HALL_EXP_RATIO` |
| 镇渊积分 | `NPC_TIER_POINTS` |
